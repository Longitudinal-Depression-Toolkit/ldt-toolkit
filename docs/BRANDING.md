# Branding and UX

LDT-Toolkit is a free and open-source project that values consistency in both visual identity and user experience. This document presents the branding and UX designs that guide the project's development and presentation.

## Branding

To maintain a consistent visual identity, all branding designs for LDT-Toolkit were created using Figma. These designs are available below for interactive visualisation.

The banner was created with [ascii-motion.app](https://ascii-motion.app).

<iframe style="border: 1px solid rgba(0, 0, 0, 0.1);" width="100%" height="460" src="../public/resources/banner.html" allowfullscreen></iframe>

If the Figma embed is blocked by your browser or network settings, open the design directly in your Figma workspace.

Feel free to open an issue on our repository for any feedback or suggestions regarding both the branding and UX designs: <https://github.com/Longitudinal-Depression-Toolkit/ldt-toolkit/issues>.
