# Data Preprocessing Tools

This page covers the preprocessing stage tools used to clean, harmonise, impute, reshape, and visualise your longitudinal dataset. Use the class docstrings below as an execution guide to understand required parameters, expected intermediate outputs, and when each transformation should be applied.

!!! tip "Execution flow"
    `fit(...)` validates and stores configuration, `preprocess(...)` runs the transformation, and `fit_preprocess(...)` executes both in sequence.
