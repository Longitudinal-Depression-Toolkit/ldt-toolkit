# MCS by LEAP

Millennium Cohort Study (MCS) reproducibility bundle by LEAP, focused on exploring whether machine learning and longitudinal-data-aware machine learning can detect children's depression trajectories from longitudinal predictors.

!!! tip "Recommended execution path"

    This preset is recommended to be run through the CLI:

    ```bash
    ldt
    ```

    You can also run it programmatically if you prefer.
